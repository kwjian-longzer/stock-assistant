## 一、VIP文章股票发现（搜索式匹配）

### 问题
旧方案用行业词典匹配关键词，本质是"猜测"而非"搜索发现"。用户要求：用每篇VIP文章的标题和简介信息，去搜索发现最符合描述的上市公司。

### 新方案：三层搜索发现

**数据源**：
1. CLS VIP API `related_stock` 字段：89篇文章中68篇有此字段，含 `[{"market":"科创板","count":1}]` 格式的板块+数量信息
2. Tushare `stock_company`：6294家公司的 `main_business`（主营业务）和 `business_scope`（经营范围）文本
3. Tushare `stock_basic`：股票名称、行业分类

**搜索流程**（每篇文章）：
```
Step 1: 提取文章搜索线索
  - 从标题/简介提取业务关键词（如"粉体材料""PCB镀铜""光模块锡粉"）
  - 读取 related_stock 字段获取板块约束（如"科创板1只"）

Step 2: 主营业务全文搜索
  - 在 stock_company.main_business 中搜索业务关键词
  - 如有板块约束，按板块前缀过滤（688=科创板, 300=创业板, 00/60=主板）
  - 模糊匹配：关键词出现在主营业务描述中即为候选

Step 3: 综合评分排序
  - 板块匹配 +3分（如related_stock说科创板，候选在科创板）
  - 主营业务包含搜索词 +5分/词
  - 股票名称包含关键词 +3分
  - 多关键词命中加分累积
  - 取Top 3作为匹配结果
```

**示例验证**：
```
文章: "算力芯片、光模块、先进封装拉动粉体材料需求"
related_stock: [{"market":"科创板","count":1}]

搜索: main_business 含 "粉体" + ts_code 以 688 开头
→ 688456.SH 有研粉材: "先进有色金属粉体材料的设计、研发、生产和销售"
→ 688116.SH 天奈科技: "碳纳米管粉体..."
→ 688786.SH 悦安新材: "羰基铁粉...粉体深加工"

有研粉材得分最高（主营业务直接是"粉体材料"），且只有1只符合"科创板1只"描述。
```

- `vip_extractor.py`: 重写 `match_stocks_by_keywords()` → `discover_stocks_by_article()`
- `fetch_data.py`: VIP采集增加分页（5页，89篇文章）

## 二、热度量化算法（按天计）

### 设计理念
用户要求：热度需要定量，按天计。参考变量：主力资金流向流量。

### 热度算法（H = 资金热度 + 信息密度 + 涨停密度）

```python
热度 H(t) = W1 * 资金流向因子 + W2 * 信息密度因子 + W3 * 涨停密度因子

# 1. 资金流向因子（权重 W1=0.5，最重要）
# 基于板块/概念内个股的 moneyflow.net_mf_amount 聚合
资金流向因子 = (板块净流入金额 / 板块总市值) * 100
# 正值=资金流入，负值=资金流出
# 连续多日流入 → 热度上升
# 主力退出（净流入转负）→ 热度下降

# 2. 信息密度因子（权重 W2=0.3）
# 基于电报/VIP文章中该热点的提及次数
信息密度 = 当日电报提及次数 + VIP文章涉及次数 * 2
# 标准化：信息密度因子 = 信息密度 / 当日所有热点平均密度

# 3. 涨停密度因子（权重 W3=0.2）
# 基于该板块涨停个股数占板块总数的比例
涨停密度 = 板块内涨停个股数 / 板块总个股数 * 100
```

### 热点生命周期判定（基于多日热度趋势）

```
天数:  Day1   Day2   Day3   Day4   Day5   Day6
热度:   20     35     55     80     72     45
趋势:   ↑      ↑     ↑↑     ↑↑     ↓      ↓↓
状态:  崛起   崛起   高潮   高潮   退烧   退烧

判定规则:
- 连续3日热度上升 + 当前H>50 → 崛起
- H>70 且当日热度≥近3日峰值 → 高潮
- 热度连续2日下降 + 资金净流出 → 退烧
- H<30 → 冷却（不纳入热点追踪）
```

### 资金流向生命周期（用户描述的过程）
```
主力开始进 → 其他资金跟进 → 主力退出 → 散户站岗

阶段1: 主力开始进
  - net_mf_amount 转正且持续3日
  - 大单买入(buy_lg_amount) > 大单卖出(sell_lg_amount)
  - 换手率开始上升

阶段2: 其他资金跟进
  - net_mf_amount 持续流入且加速
  - 中单买入增加
  - 涨停个股增加

阶段3: 主力退出
  - net_mf_amount 由正转负
  - 大单净流出（sell_lg > buy_lg）
  - 但散户小单仍在买入（buy_sm > sell_sm）

阶段4: 散户站岗
  - net_mf_amount 持续为负
  - 股价下跌
  - 换手率下降
```

### 数据需求
- `moneyflow`（Tushare）: 个股资金流向，按日 → 聚合到板块
- `daily`（Tushare）: 涨跌幅、成交量
- `daily_basic`（Tushare）: 换手率
- 电报归档: 信息密度（按热点关键词统计）

### 热度曲线呈现（报告中的文本可视化）
